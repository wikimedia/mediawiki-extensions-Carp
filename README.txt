== What? ==

The Carp extension enables template call stack backtracing and error reporting for parser functions.
(Error reporting for templates is implemented in another extension named TemplateSpecial.)

== Documentation ==

*   Extension page (address below in `Links' section),
*   Ebedded documentation in `Carp.php' file, or
*   Run `doxygen' to generate documentation in HTML format.
